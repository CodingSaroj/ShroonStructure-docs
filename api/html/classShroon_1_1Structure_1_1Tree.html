template<typename ValueType >
std::vector<TreeNode<ValueType> *> Shroon::Structure::Tree< ValueType >::GetNodesAtHeight ( size_t  height)
inline

Returns an array of pointer to all the nodes at a certain height in the tree.

Parameters
heightThe height from which nodes are returned.
Returns
An std::vector of pointer to all the nodes at a certain height in the tree. An array of size 1 means no nodes are on that height. The array also has some nullptr elements which separate children of on node from children of other nodes.

Example - In the following tree:

A
/ \
B C
/\ /\
D E F G

If we retrieve nodes at height 2 we expect [D, E, F, G] but we get [nullptr, D, E, nullptr, F, G]. The nullptr means the node afterwards are children of the same parent and when another nullptr comes it means that the parent of the following nodes are different than the previous one.

In this case:

  • 0: nullptr - Following nodes are children of first parent node (B).
  • 1: D - Child of one node (B).
  • 2: E - Child of one node (B).
  • 3: nullptr - Following nodes are children of second parent node (C).
  • 4: F - Child of another node (C).
  • 5: G - Child of another node (C).
